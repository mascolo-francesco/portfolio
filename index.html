Droven AI Prompt Workspace

Advanced workspace for professional prompt engineers

A complete web environment with library management, versioning, and a dedicated AI assistant to improve the entire lifecycle of prompts.

STACK: HTML5 · CSS3 · JavaScript · Appwrite · GroqCloud

Case Study

Droven is an AI workspace that brings together prompt libraries, version control, and an intelligent assistant with automation features. The interface follows a glass morphism style inspired by the Onyx Dark theme.

Key capabilities
  • Prompt library with nested folders, tagging, and drag & drop.
  • AI enhancer with multi-stage pipelines, specialized profiles, and professional diffing.
  • Droven Engineer assistant with function calling for quick actions.
  • Automatic version control with restore, statistics, and diff.
Experience & platform
  • Responsive UI with desktop/mobile support and keyboard navigation.
  • Full internationalization (Italian/English) via i18next.
  • Secure Appwrite authentication with backup and cloud sync.
  • Python serverless functions to orchestrate AI calls on GroqCloud.

Status: Ongoing — roadmap includes light theme, team collaboration, and plugins.
